Quiz Game

Simple console quiz game
by Karol Szpek

Instructions

One point is added every single time player guesses the right answer, no points are substracted in case of a mistake. Questions may repeat from time to time. Player can quit the game when writing 'exit' in the input box.

1. Open browser's console (Ctrl+Shift+K).
2. Press the button when you're ready.

About the project

Development

Game's script was written in pure Javascript, without using any JS framework. The game function from external script.js file is run after user presses the button. The questions were created using function constuctor. The game consists of ten different questions chosen randomly from a question array.
